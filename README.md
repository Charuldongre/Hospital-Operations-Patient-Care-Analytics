# 🏥 Hospital Operations & Patient Care Analytics

## 📌 Project Overview

This project analyzes hospital operations, patient demographics, treatment costs, department performance, and patient outcomes using **Python, SQL, and Power BI**.

The objective is to transform raw healthcare data into meaningful business insights that support better decision-making, optimize resource allocation, reduce readmission rates, and improve patient care.

---

# 🎯 Business Problem

Hospitals generate large volumes of operational and patient data every day. However, identifying high-cost diseases, monitoring readmission trends, improving patient satisfaction, and allocating resources efficiently can be challenging without proper data analysis.

This project aims to answer key business questions such as:

- Which diseases incur the highest treatment costs?
- Which departments have the highest patient volume?
- Which diseases have the highest readmission rates?
- Which departments provide the highest patient satisfaction?
- How does hospital stay impact treatment cost?

# 🔍 Key Insights

- Cancer generated the highest treatment costs.
- Oncology department incurred the highest expenditure.
- Longer hospital stays significantly increased treatment costs.
- Readmission rates varied considerably across diseases.
- Patient satisfaction differed among departments.

---

# 💡 Recommendations

- Reduce readmissions through improved follow-up care.
- Optimize resource allocation for high-cost diseases.
- Improve patient satisfaction in lower-performing departments.
- Monitor long-stay patients more effectively.
